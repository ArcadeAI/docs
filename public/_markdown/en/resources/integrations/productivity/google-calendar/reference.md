# GoogleCalendar Reference

Below is a reference of enumerations used by some tools in the GoogleCalendar MCP Server:

## EventVisibility

-   **DEFAULT**: `default`
-   **PUBLIC**: `public`
-   **PRIVATE**: `private`
-   **CONFIDENTIAL**: `confidential`

## SendUpdatesOptions

-   **NONE**: `none`
-   **ALL**: `all`
-   **EXTERNAL\_ONLY**: `externalOnly`

## UpdateGoogleMeetOptions

-   **NONE**: `none`
-   **ADD**: `add`
-   **REMOVE**: `remove`
