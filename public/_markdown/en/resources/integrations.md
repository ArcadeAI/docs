# MCP Servers

There are 5 designations for Arcade MCP Servers:

## Arcade Optimized

Official MCP Servers hand-crafted by Arcade that are ready for use and optimized for LLM-usage. [Learn more](/guides/create-tools/improve/types-of-tools.md#optimized-tools).

## Arcade Unoptimized

Unoptimized (auto-generated) MCP Servers developed by Arcade that may require customization. [Learn more](/guides/create-tools/improve/types-of-tools.md#unoptimized-tools).

## Verified

Community-created MCP Servers, thoroughly tested and verified by Arcade.

## Community

Created and maintained by the Arcade community, offering a wide range of MCP Servers.

## Auth Provider

Auth integrations allow you to develop custom tools that connect your agent APIs and services.

## Build your own MCP Server

Don't see what you need? Use Arcade's SDK to integrate with any service or API.

[Learn how to build a MCP Server](/guides/create-tools/tool-basics/build-mcp-server.md)
