# The Arcade Registry

The **Arcade Registry** is how agentic tool developers share their work with the world. Arcade.dev is collecting all the integrations that  will ever need in one place - think HuggingFace or Pypi but for LLM . A powerful community of open source and for-profit developers building out robust and evaluated workflows is how agents can be elevated to being _truly_ useful.
