### Install and use Arcade

<Steps>

### Obtain an API key

Install the Arcade CLI and SDK and log in. Your Arcade API key will be printed to the console as well as saved to `~/.arcade/credentials.yaml`.

```bash
pip install 'arcade-ai[fastapi]'
arcade login
```

<Tip>Obtaining an API key through the Arcade dashboard is coming soon!</Tip>

### Install the SDK

Fire up your terminal and run:

```bash
pip install arcadepy
```

### Set up your Arcade client

In your python application, import the Arcade client and initialize it with your API key:

```python
from arcadepy import Arcade

client = Arcade(
    api_key=<ARCADE_API_KEY>
)
```

### Call a tool

You can now call tools with your client. Here's an example of calling the `Math.Sqrt` tool:

```python
response = client.tools.execute(
    tool_name="Math.Sqrt",
    input={"a": 625},
    user_id="user@example.com",
)
print(response.output.value)  # 25
```

For a complete list of available tools, visit our [Toolkits page](/toolkits). You can explore tools for email, calendar, GitHub, and many other services.
