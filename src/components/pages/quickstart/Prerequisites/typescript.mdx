### Install and use our TypeScript SDK

<Steps>

### Install the SDK

Fire up your terminal and run:

```bash
npm install @arcadeai/arcadejs
```

### Set up your Arcade client

```typescript
import Arcade from "@arcadeai/arcadejs";

const client = new Arcade({
  apiKey: process.env["ARCADE_API_KEY"],
});
```

### Call a tool

You can now call tools with your client. Here's an example of calling the `Google.ListEmails` tool:

```typescript
async function main() {
  const executeToolResponse = await client.tools.execute({
    tool_name: "Google.ListEmails",
    inputs: { n_emails: 10 },
    user_id: "user@example.com",
  });

  console.log(executeToolResponse.id);
}

main();
```

For a complete list of available tools, visit our [Toolkits page](/toolkits). You can explore tools for email, calendar, GitHub, and many other services.
