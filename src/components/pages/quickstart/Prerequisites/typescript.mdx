### Install and use Arcade

<Typescript>

### Install the Arcade TypeScript / JavaScript Client

Fire up your terminal and run:

```bash
npm install @arcadeai/arcadejs
```

### Set your Arcade API key

<SetApiKey />

### Set up your Arcade client

In your JavaScript / TypeScript application, import the Arcade client and initialize it:

```typescript
import Arcade from "@arcadeai/arcadejs";

const client = new Arcade();
```

Alternatively, you can initialize the client passing your API key as an argument, but it is not recommended to hardcode sensitive information:

```typescript
const client = new Arcade({apiKey: "your_api_key_here"});
```
