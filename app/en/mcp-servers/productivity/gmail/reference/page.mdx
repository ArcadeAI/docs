# Gmail Reference

Below is a reference of enumerations used by some tools in the Gmail MCP Server:

## GmailReplyToWhom

- **EVERY_RECIPIENT**: `every_recipient`
- **ONLY_THE_SENDER**: `only_the_sender`

## DateRange

- **TODAY**: `today`
- **YESTERDAY**: `yesterday`
- **LAST_7_DAYS**: `last_7_days`
- **LAST_30_DAYS**: `last_30_days`
- **THIS_MONTH**: `this_month`
- **LAST_MONTH**: `last_month`
- **THIS_YEAR**: `this_year`
