## Auth

The Arcade Airtable MCP Server uses the [Airtable auth provider](/home/auth-providers/airtable) to connect to users' Airtable accounts.

With the Arcade Cloud Platform, there's nothing to configure. Your users will see `Arcade` as the name of the application that's requesting permission.

With a self-hosted installation of Arcade, you need to [configure the Airtable auth provider](/home/auth-providers/airtable#configuring-airtable-auth) with your own Airtable app credentials.
