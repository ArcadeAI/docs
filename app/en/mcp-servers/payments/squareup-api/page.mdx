The SquareupApi MCP Server provides a comprehensive set of tools for interacting with Squareâ€™s APIs. These tools enable building agents and apps to:

- Manage authentication and token status.
- Access and manage bank accounts, cards, payouts, and payment/refund records.
- Create, retrieve, cancel, and manage payments, payouts, refunds, and disputes (including dispute evidence).
- Work with the Catalog, inventory, invoices, orders, and related custom attributes.
- Manage customers, customer groups/segments, loyalty programs, promotions, rewards, and customer-specific custom attributes.
- Manage locations, location settings, devices, device codes, and merchant-level settings and custom attributes.
- List and manage bookings, booking profiles, appointments custom attributes, and booking-related actions.
- Manage subscriptions, scheduled actions, and subscription events.
- Retrieve and manage terminal actions, checkouts, and terminal refunds (including cancel/dismiss actions).
- Manage gift cards and gift card activities.
- Manage employees, jobs, timecards, shifts, wages, break types, and related labor data.
- List and manage webhooks (event types, subscriptions) and control event searchability.
- Retrieve Square Online site snippets and manage payment links.
- Retrieve vendor, merchant, and seller details and custom attributes.

Use these endpoints to query, list, retrieve detailed records, delete or cancel resources, and manage custom attributes across Square entities.
