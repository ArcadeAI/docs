# Reference

Below is a reference of enumerations used by some of the tools in the Brightdata MCP Server:

- **GOOGLE**: `google`
- **BING**: `bing`
- **YANDEX**: `yandex`

- **IMAGES**: `images`
- **SHOPPING**: `shopping`
- **NEWS**: `news`

- **IPAD**: `ipad`
- **ANDROID**: `android`
- **ANDROID_TABLET**: `android_tablet`
