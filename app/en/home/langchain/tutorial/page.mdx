# LangChain/LangGraph with Arcade

Build a production-ready agent that can automatically choose and execute tools based on user intent.

## What you'll build

A morning briefing agent that:
- Checks Gmail for unread messages
- Lists today's calendar events
- Sends a summary to Slack
- Uses LangGraph for reliable agent orchestration

**Time to complete:** 15 minutes

## Full example

<Tabs items={["Python", "TypeScript"]}>

<Tabs.Tab>

```python
from arcadepy import Arcade
from langchain_openai import ChatOpenAI
from langgraph.prebuilt import create_react_agent

# Initialize
arcade = Arcade(api_key="your_api_key_here")
user_id = "user@example.com"

# Get tools
tools = [
    arcade.get_tool("Gmail.ListMessages", user_id=user_id),
    arcade.get_tool("GoogleCalendar.ListEvents", user_id=user_id),
    arcade.get_tool("Slack.SendMessage", user_id=user_id),
]

# Create agent
llm = ChatOpenAI(model="gpt-4o")
agent = create_react_agent(llm, tools)

# Run agent
response = agent.invoke({
    "messages": [
        {
            "role": "user",
            "content": """Generate my morning briefing:
            1. Check Gmail for unread messages from today
            2. Check calendar for today's events
            3. Send summary to #general on Slack
            """
        }
    ]
})

print(response["messages"][-1].content)
```

</Tabs.Tab>

<Tabs.Tab>

```typescript
import Arcade from "@arcadeai/arcadejs";
import { ChatOpenAI } from "@langchain/openai";
import { createReactAgent } from "@langchain/langgraph/prebuilt";

// Initialize
const arcade = new Arcade({
  apiKey: "your_api_key_here",
});
const userId = "user@example.com";

// Get tools
const tools = [
  await arcade.getTool("Gmail.ListMessages", { userId }),
  await arcade.getTool("GoogleCalendar.ListEvents", { userId }),
  await arcade.getTool("Slack.SendMessage", { userId }),
];

// Create agent
const llm = new ChatOpenAI({ model: "gpt-4o" });
const agent = createReactAgent({ llm, tools });

// Run agent
const response = await agent.invoke({
  messages: [
    {
      role: "user",
      content: `Generate my morning briefing:
      1. Check Gmail for unread messages from today
      2. Check calendar for today's events
      3. Send summary to #general on Slack
      `,
    },
  ],
});

console.log(response.messages[response.messages.length - 1].content);
```
