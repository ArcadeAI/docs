# Generic Python with Arcade

Use Arcade's SDK directly in any Python application without a specific agent framework.

## What you'll build

A simple Python script that:
- Fetches Gmail messages
- Lists calendar events
- Sends a Slack notification
- No framework required - just the Arcade SDK

**Time to complete:** 15 minutes

## Prerequisites

- Python 3.8 or higher
- [Arcade API key](/home/api-keys)
- Gmail, Google Calendar, and Slack accounts

## Step 1: Install the Arcade SDK

```bash
pip install arcadepy
```

## Step 2: Initialize the client

Create a new file `briefing.py`:

```python
from arcadepy import Arcade

# Initialize the Arcade client
client = Arcade(api_key="your_api_key_here")

# Your user identifier
user_id = "user@example.com"
```

## Step 3: Call tools directly

```python
# Check Gmail
gmail_response = client.tools.execute(
    tool_name="Gmail.ListMessages",
    input={"maxResults": 10, "q": "is:unread"},
    user_id=user_id
)

print(f"Unread emails: {len(gmail_response.output.value)}")

# Check Calendar
calendar_response = client.tools.execute(
    tool_name="GoogleCalendar.ListEvents",
    input={
        "calendarId": "primary",
        "timeMin": "2024-01-01T00:00:00Z",
        "maxResults": 10
    },
    user_id=user_id
)

print(f"Upcoming events: {len(calendar_response.output.value)}")

# Send Slack message
slack_response = client.tools.execute(
    tool_name="Slack.SendMessage",
    input={
        "channel": "#general",
        "text": "Morning briefing: 5 unread emails, 3 meetings today"
    },
    user_id=user_id
)

print("Briefing sent to Slack!")
```

## Step 4: Handle authentication

If a tool requires authentication:

```python
# Check if auth is needed
auth_response = client.tools.authorize(
    tool_name="Gmail.ListMessages",
    user_id=user_id
)

if auth_response.status != "completed":
    print(f"Please authorize: {auth_response.url}")
    # Wait for user to complete auth
    client.auth.wait_for_completion(auth_response.id)
```

## Step 5: Run your script

```bash
python briefing.py
```

## Expected output

```
Unread emails: 5
Upcoming events: 3
Briefing sent to Slack!
```
