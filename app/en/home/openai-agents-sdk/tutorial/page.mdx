# OpenAI Agents SDK with Arcade

Build production-ready agents using OpenAI's official Agents SDK with Arcade's 100+ tools.

## What you'll build

A morning briefing agent that:
- Checks Gmail for unread messages
- Lists today's calendar events
- Sends a summary to Slack
- Uses OpenAI's Agents SDK for reliable orchestration

**Time to complete:** 15 minutes

## Full example

<Tabs items={["Python", "TypeScript"]}>

<Tabs.Tab>

```python
from openai_agents import Agent
from arcadepy import Arcade

# Initialize
arcade = Arcade(api_key="your_arcade_api_key")
user_id = "user@example.com"

# Get tools
tools = [
    arcade.get_tool("Gmail.ListMessages", user_id=user_id),
    arcade.get_tool("GoogleCalendar.ListEvents", user_id=user_id),
    arcade.get_tool("Slack.SendMessage", user_id=user_id),
]

# Create agent
agent = Agent(
    name="Morning Briefing Agent",
    instructions="Generate morning briefings from emails and calendar.",
    model="gpt-4o",
    tools=tools,
    api_key="your_openai_api_key"
)

# Run agent
response = agent.run("""Generate my morning briefing:
1. Check Gmail for unread messages from today
2. Check calendar for today's events
3. Send summary to #general on Slack
""")

print(response.content)
```

</Tabs.Tab>

<Tabs.Tab>

```typescript
import { Agent } from "openai-agents-sdk";
import Arcade from "@arcadeai/arcadejs";

// Initialize
const arcade = new Arcade({
  apiKey: "your_arcade_api_key",
});
const userId = "user@example.com";

// Get tools
const tools = [
  await arcade.getTool("Gmail.ListMessages", { userId }),
  await arcade.getTool("GoogleCalendar.ListEvents", { userId }),
  await arcade.getTool("Slack.SendMessage", { userId }),
];

// Create agent
const agent = new Agent({
  name: "Morning Briefing Agent",
  instructions: "Generate morning briefings from emails and calendar.",
  model: "gpt-4o",
  tools,
  apiKey: "your_openai_api_key",
});

// Run agent
const response = await agent.run(`Generate my morning briefing:
1. Check Gmail for unread messages from today
2. Check calendar for today's events
3. Send summary to #general on Slack
`);

console.log(response.content);
```

</Tabs.Tab>

</Tabs>
