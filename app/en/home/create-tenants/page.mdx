# Create Tenants

This is a placeholder page for creating tenants in Arcade.

## Overview

Learn how to create and manage tenants for multi-user applications.

## What are Tenants?

Tenants allow you to isolate data and configurations for different organizations or user groups within your application.

## Creating a Tenant

1. Navigate to the Dashboard
2. Go to the Tenants section
3. Click "Create New Tenant"
4. Configure tenant settings
5. Set up authentication

## Tenant Configuration

- Tenant name and description
- Authentication settings
- Tool access permissions
- Usage limits and quotas
