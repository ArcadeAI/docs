El kit de herramientas SlackApi ofrece un conjunto completo de herramientas para administrar espacios de trabajo de Slack, automatizar mensajería, gestionar canales, llamadas, archivos, emojis, grupos de usuarios, invitaciones y datos de usuarios/equipos. Las capacidades clave incluyen:

- Gestión de espacios de trabajo y equipos: crear/actualizar nombre/descripción del equipo, obtener información del equipo, configuraciones, preferencias, logs de integración, usuarios facturables y listar/inspeccionar equipos en una organización Enterprise.
- Operaciones de usuarios e identidad: listar usuarios del espacio de trabajo/equipo, encontrar usuarios por email, obtener perfiles de usuarios, presencia, identidad y gestionar fotos de perfil.
- Canales y conversaciones: crear/unirse a conversaciones, obtener información y miembros de conversaciones, listar canales/conversaciones accesibles a un usuario, abrir/reanudar DMs, invitar usuarios, establecer cursores de lectura y gestionar invitaciones de canales compartidos.
- Mensajería y programación: enviar mensajes (regulares y efímeros), programar/eliminar mensajes programados, listar mensajes programados, obtener enlaces permanentes de mensajes y buscar mensajes/archivos.
- Llamadas: registrar llamadas, obtener información de llamadas, agregar/eliminar participantes.
- Archivos y compartir: obtener URLs de carga externa, obtener información de archivos remotos, compartir archivos remotos a canales, habilitar compartir público.
- Marcadores, pines y reacciones: agregar/editar/eliminar marcadores, anclar/listar elementos anclados, agregar/eliminar reacciones.
- Gestión de emojis: listar emojis personalizados, renombrar emojis y agregar alias de emojis (Enterprise).
- Grupos de usuarios: crear, habilitar/deshabilitar, listar, actualizar grupos de usuarios y su membresía.
- Gestión de invitaciones/flujos de trabajo: listar invitaciones de espacio de trabajo pendientes/aprobadas/denegadas, aceptar/aprobar/denegar invitaciones de canales compartidos y listar invitaciones compartidas.
- Herramientas de administración y verificación: obtener configuraciones del espacio de trabajo, propietarios, canales para grupos de usuarios org, listar emojis/equipos enterprise, revocar tokens, verificar código de llamada API y recuperar logs de integración.
- Comportamiento personalizado: proporcionar unfurling personalizado para URLs.

Este kit de herramientas está diseñado para administradores y aplicaciones que requieren acceso amplio a la API de Slack (scopes de admin, invites, calls, chat, files, usergroups, reactions, users).

- **crop_box_size** (`string`, opcional) El tamaño del cuadro de recorte cuadrado para la foto de perfil en píxeles. Especificar el ancho y alto, que son el mismo valor para un cuadrado.
- **crop_box_x_coordinate** (`string`, opcional) Coordenada X de la esquina superior izquierda del cuadro de recorte para la foto de perfil.
- **crop_y_coordinate** (`string`, opcional) Coordenada Y de la esquina superior izquierda del cuadro de recorte para la foto de perfil del usuario en Slack. Esto determina dónde comenzará el recorte de la imagen en el eje vertical.
- **profile_photo_image** (`string`, opcional) El archivo de imagen a establecer como foto de perfil de Slack. Proporcionar datos de imagen directamente con el tipo de contenido correcto (ej., image/jpeg, image/png).
