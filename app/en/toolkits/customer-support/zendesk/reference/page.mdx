# Zendesk Reference

Below is a reference of enumerations used by some tools in the Zendesk toolkit:

## TicketStatus

- **NEW**: `new`
- **OPEN**: `open`
- **PENDING**: `pending`
- **SOLVED**: `solved`
- **CLOSED**: `closed`

## SortOrder

- **ASC**: `asc`
- **DESC**: `desc`

## ArticleSortBy

- **CREATED_AT**: `created_at`
- **RELEVANCE**: `relevance`
