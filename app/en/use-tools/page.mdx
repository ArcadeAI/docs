# Use Arcade Tools

Choose how you want to use Arcade's 100+ pre-built integrations.

## Two Paths

### In Your IDE / MCP Client

Add tools directly to your code editor or AI assistant.

- **Best for:** Developers who want AI tools while coding
- **Setup time:** 2-5 minutes
- **Examples:** Cursor, VS Code, Claude Desktop

[Get started with IDE setup →](/use-tools/ide)

---

### In Your Agent / Application

Integrate tools into your own AI-powered applications.

- **Best for:** Developers building apps with AI capabilities
- **Setup time:** 15-30 minutes
- **Examples:** LangChain agents, OpenAI Agents SDK, custom apps

[Get started with agent integration →](/use-tools/agent)
