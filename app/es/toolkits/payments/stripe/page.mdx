## Stripe.CreateCustomer

Crea un cliente en Stripe.

**Parámetros**

- **`name`** _(string, requerido)_: El nombre del cliente.
- **`email`** _(string, opcional)_: La dirección de correo electrónico del cliente.

## Stripe.ListCustomers

Obtén una lista de clientes de Stripe.

**Parámetros**

- **`limit`** _(int, opcional, por defecto 10)_: Un límite en el número de objetos a devolver. El límite puede variar entre 1 y 100.
- **`email`** _(string, opcional)_: Un filtro sensible a mayúsculas en la lista basado en el campo de correo electrónico del cliente.

## Auth

El kit de herramientas de Arcade Stripe utiliza el [Stripe Agent Toolkit](https://github.com/stripe/agent-toolkit) para interactuar con la API de Stripe.

- **Secreto Requerido:**
  - `STRIPE_SECRET_KEY`: Tu clave API de Stripe.
