El kit de herramientas de CRM de Salesforce de Arcade proporciona un conjunto preconstruido de herramientas para interactuar con Cuentas, Leads, Contactos, etc. en el CRM de Salesforce. Estas herramientas facilitan la creación de agentes y aplicaciones de IA que pueden:

- Buscar Cuentas y Contactos por palabras clave o recuperarlos por ID
- Leer información sobre Cuentas, como metadatos de la empresa, oportunidades, tratos, etc.
- Leer información sobre Contactos, como nombre, direcciones de correo electrónico, números de teléfono, mensajes de correo electrónico, registros de llamadas, notas, reuniones, tareas, etc.
- Crear contactos

## Instalar y Ejecutar el Motor de Arcade

En este momento, necesitas [auto-alojar](/home/auth-providers/salesforce) el Motor de Arcade para usar el kit de herramientas de Salesforce. Sigue las instrucciones paso a paso en la página de [Proveedor de Autenticación de Salesforce](/home/auth-providers/salesforce).

## Instalar

```bash
pip install arcade_salesforce
```

Busca cuentas en Salesforce y las devuelve con información relacionada: contactos, leads, notas, llamadas, oportunidades, tareas, correos electrónicos y eventos (hasta 10 elementos de cada tipo).

**Parámetros**

- **query** _(string, requerido)_ La consulta para buscar cuentas. DEBE tener más de un carácter. Coincidirá las palabras clave con todos los campos de la cuenta, como nombre, sitio web, teléfono, dirección, etc. Ej. 'Acme'.
- **limit** _(int, opcional, por defecto `10`)_ El número máximo de cuentas a devolver. Por defecto es 10. El máximo permitido es 10.
- **page** _(string, opcional)_ El número de página a devolver. Por defecto es 1 (primera página de resultados)."

Crea un contacto en Salesforce asociado con una cuenta.

**Parámetros**

- **account_id** _(string, requerido)_ El ID de la cuenta para la que crear el contacto.
- **first_name** _(string, requerido)_ El nombre del contacto.
- **last_name** _(string, requerido)_ El apellido del contacto.
- **email** _(string, requerido)_ La dirección de correo electrónico del contacto.
- **phone** _(string, opcional)_ El número de teléfono del contacto.
- **mobile_phone** _(string, opcional)_ El número de teléfono móvil del contacto.
- **title** _(string, opcional)_ El título del contacto. Ej. 'CEO', 'Director de Ventas', 'CTO', etc.
- **department** _(string, opcional)_ El departamento del contacto. Ej. 'Marketing', 'Ventas', 'IT', etc.
- **description** _(string, opcional)_ Una descripción del contacto.

## Auto-alojamiento del Motor de Arcade con Autenticación de Salesforce

En este momento, Arcade Cloud no soporta la autenticación de Salesforce.

Para usar el kit de herramientas de Salesforce (o desarrollar herramientas personalizadas para Salesforce), debes [auto-alojar el Motor de Arcade](/home/local-deployment/install/local) y [configurar el proveedor de autenticación de Salesforce](/home/auth-providers/salesforce) en la configuración de tu motor.
