## Hubspot.CreateEmailActivity

<br />
<TabbedCodeBlock
  tabs={[
    {
      label: "Llamar a la Herramienta Directamente",
      content: {
        Python: [
          "/examples/integrations/toolkits/hubspot/create_email_activity_example_call_tool.py",

/>

Crear una actividad de compromiso de correo electrónico registrada con campos esenciales, incluyendo encabezados de correo electrónico.

**Parámetros**

- **subject** (`string`, requerido) Asunto del correo electrónico.
- **when_occurred** (`string`, requerido) Cuándo ocurrió el correo electrónico (formato de fecha ISO: YYYY-MM-DDTHH:MM:SS).
- **from_email** (`string`, requerido) Dirección de correo electrónico del remitente.
- **to_email** (`string`, requerido) Dirección de correo electrónico del destinatario principal.
- **body_text** (`string`, opcional) Cuerpo del correo electrónico en texto plano.
- **body_html** (`string`, opcional) Cuerpo del correo electrónico en formato HTML.
- **from_first_name** (`string`, opcional) Nombre del remitente.
- **from_last_name** (`string`, opcional) Apellido del remitente.
- **to_first_name** (`string`, opcional) Nombre del destinatario principal.
- **to_last_name** (`string`, opcional) Apellido del destinatario principal.
- **cc_emails** (`array[string]`, opcional) Direcciones de correo electrónico de los destinatarios en copia.
- **bcc_emails** (`array[string]`, opcional) Direcciones de correo electrónico de los destinatarios en copia oculta.
- **direction** (`Enum` [HubspotEmailDirection](/toolkits/sales/hubspot/reference#HubspotEmailDirection), opcional) Dirección en la que se envió el correo electrónico (EMAIL, INCOMING_EMAIL, FORWARDED_EMAIL).
- **status** (`Enum` [HubspotEmailStatus](/toolkits/sales/hubspot/reference#HubspotEmailStatus), opcional) Estado del correo electrónico que indica el estado del correo.
- **associate_to_contact_id** (`integer`, opcional) ID de contacto para asociar este correo electrónico.
- **associate_to_company_id** (`integer`, opcional) ID de empresa para asociar este correo electrónico.
- **associate_to_deal_id** (`integer`, opcional) ID de trato para asociar este correo electrónico.
