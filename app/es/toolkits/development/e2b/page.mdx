El kit de herramientas Arcade E2B proporciona un conjunto predefinido de herramientas para ejecutar código en un entorno aislado. Estas herramientas facilitan la creación de agentes y aplicaciones de IA que pueden:

- Ejecutar código en un entorno aislado
- Crear un gráfico estático de matplotlib

## E2b.RunCode

<br />
<TabbedCodeBlock
  tabs={[
    {
      label: "Llamar a la Herramienta Directamente",
      content: {
        Python: [
          "/examples/integrations/toolkits/e2b/run_code_example_call_tool.py",
        ],
        JavaScript: [
          "/examples/integrations/toolkits/e2b/run_code_example_call_tool.js",
        ],
      },
    },
  ]}
/>

Ejecutar código en un entorno aislado y devolver la salida.

**Autenticación:**

- **Variables de Entorno Requeridas:**
  - `E2B_API_KEY`: Tu clave API para autenticación.

**Parámetros**

- **`code`** _(string, requerido)_ El código a ejecutar.
- **`language`** _(string, opcional)_ El lenguaje del código. Los valores válidos son 'python', 'js', 'r', 'java', 'bash'. Por defecto es 'python'.

## E2b.CreateStaticMatplotlibChart

<br />
<TabbedCodeBlock
  tabs={[
    {
      label: "Llamar a la Herramienta Directamente",
      content: {
        Python: [
          "/examples/integrations/toolkits/e2b/create_static_matplotlib_chart_example_call_tool.py",
        ],
        JavaScript: [
          "/examples/integrations/toolkits/e2b/create_static_matplotlib_chart_example_call_tool.js",
        ],
      },
    },
  ]}
/>

Ejecutar el código Python proporcionado para generar un gráfico estático de matplotlib. El gráfico resultante se devuelve como una imagen codificada en base64.

**Autenticación:**

- **Variables de Entorno Requeridas:**
  - `E2B_API_KEY`: Tu clave API de [E2B](https://e2b.dev/) para autenticación.

**Parámetros**

- **`code`** _(string, requerido)_ El código Python a ejecutar.

## Autenticación

El kit de herramientas Arcade E2B utiliza [E2B](https://e2b.dev/) para ejecutar código en un entorno aislado.

**Variables de Entorno Globales:**

- `E2B_API_KEY`: Tu clave API de [E2B](https://e2b.dev/).
