# Referencia para el Toolkit Firecrawl

## Formatos

El formato de la página web extraída.

- **`MARKDOWN`**: Devuelve la página web extraída en formato Markdown.
- **`HTML`**: Devuelve la página web extraída en formato HTML.
- **`RAW_HTML`**: Devuelve el HTML sin procesar de la página web extraída.
- **`LINKS`**: Devuelve solo los enlaces de la página web extraída.
- **`SCREENSHOT`**: Toma una captura de pantalla de la página web extraída.
- **`SCREENSHOT_AT_FULL_PAGE`**: Toma una captura de pantalla de toda la página web extraída.
