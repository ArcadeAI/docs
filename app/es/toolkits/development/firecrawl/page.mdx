El kit de herramientas Arcade Firecrawl proporciona un conjunto preconstruido de herramientas para interactuar con sitios web. Estas herramientas facilitan la creación de agentes y aplicaciones de IA que pueden:

- Raspar páginas web
- Rastrear sitios web
- Mapear estructuras de sitios web
- Recuperar el estado y los datos del rastreo
- Cancelar rastreos en curso

## Firecrawl.ScrapeUrl

<br />

        ],

/>

Raspar una URL y devolver datos en formatos especificados.

**Autenticación:**

- **Variables de Entorno Requeridas:**
  - `FIRECRAWL_API_KEY`: Tu clave API de [Firecrawl](https://www.firecrawl.dev/).

**Parámetros**

- **`url`** _(string, requerido)_ La URL a raspar.
- **`formats`** _(enum ([Formats](/toolkits/development/firecrawl/reference#formats)), opcional)_ El formato de la página web raspada. Por defecto es `Formats.MARKDOWN`.
- **`only_main_content`** _(bool, opcional)_ Solo devolver el contenido principal de la página. Por defecto es `True`.
- **`include_tags`** _(list, opcional)_ Lista de etiquetas a incluir en la salida.
- **`exclude_tags`** _(list, opcional)_ Lista de etiquetas a excluir de la salida.
- **`wait_for`** _(int, opcional)_ Retraso en milisegundos antes de obtener el contenido. Por defecto es `10`.
- **`timeout`** _(int, opcional)_ Tiempo de espera en milisegundos para la solicitud. Por defecto es `30000`.

## Firecrawl.CrawlWebsite

<br />
<TabbedCodeBlock
  tabs={[
    {
      label: "Llamar a la Herramienta Directamente",
      content: {
        Python: [
          "/examples/integrations/toolkits/firecrawl/crawl_website_example_call_tool.py",
        ],
        JavaScript: [
          "/examples/integrations/toolkits/firecrawl/crawl_website_example_call_tool.js",
        ],
      },
    },
  ]}
/>

Rastrear un sitio web y devolver el estado y los datos del rastreo.

**Autenticación:**

- **Variables de Entorno Requeridas:**
  - `FIRECRAWL_API_KEY`: Tu clave API de [Firecrawl](https://www.firecrawl.dev/).

**Parámetros**

- **`url`** _(string, requerido)_ La URL a rastrear.
- **`exclude_paths`** _(list, opcional)_ Patrones de URL a excluir del rastreo.
- **`include_paths`** _(list, opcional)_ Patrones de URL a incluir en el rastreo.
- **`max_depth`** _(int, requerido)_ Profundidad máxima para rastrear. Por defecto es `2`.
- **`ignore_sitemap`** _(bool, requerido)_ Ignorar el mapa del sitio web. Por defecto es `True`.
- **`limit`** _(int, requerido)_ Limitar el número de páginas a rastrear. Por defecto es `10`.
- **`allow_backward_links`** _(bool, requerido)_ Permitir la navegación a páginas vinculadas anteriormente. Por defecto es `False`.
- **`allow_external_links`** _(bool, requerido)_ Permitir seguir enlaces a sitios web externos. Por defecto es `False`.
- **`webhook`** _(string, opcional)_ URL para enviar una solicitud POST cuando el rastreo se inicie, actualice y complete.
- **`async_crawl`** _(bool, requerido)_ Ejecutar el rastreo de forma asíncrona. Por defecto es `True`.

<br />

Mapear un sitio web desde una única URL a un mapa de todo el sitio web.

**Autenticación:**

- **Variables de Entorno Requeridas:**
  - `FIRECRAWL_API_KEY`: Tu clave API de [Firecrawl](https://www.firecrawl.dev/).

**Parámetros**

- **`url`** _(string, requerido)_ La URL base desde la cual comenzar a rastrear.
- **`search`** _(string, opcional)_ Consulta de búsqueda a utilizar para el mapeo.
- **`ignore_sitemap`** _(bool, requerido)_ Ignorar el mapa del sitio web. Por defecto es `True`.
- **`include_subdomains`** _(bool, requerido)_ Incluir subdominios del sitio web. Por defecto es `False`.
- **`limit`** _(int, requerido)_ Número máximo de enlaces a devolver. Por defecto es `5000`.

## Autenticación

El kit de herramientas Web de Arcade utiliza [Firecrawl](https://www.firecrawl.dev/) para raspar, rastrear y mapear sitios web.

**Variables de Entorno Globales:**

- `FIRECRAWL_API_KEY`: Tu clave API de [Firecrawl](https://www.firecrawl.dev/).
