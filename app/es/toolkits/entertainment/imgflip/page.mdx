Buscar plantillas de memes por consulta (función Premium)

Esta herramienta busca en la base de datos de Imgflip de más de 1 millón de plantillas de memes
para encontrar las que coincidan con tu consulta de búsqueda. Esta es una función Premium que
requiere una suscripción paga a Imgflip.

**Parámetros**

- `query` _(string, requerido)_ Consulta de búsqueda para encontrar plantillas de memes. Sé específico para obtener mejores resultados.
- `include_nsfw` _(booleano, opcional)_ Incluir memes no seguros para el trabajo en los resultados de búsqueda. Por defecto es False.
- `limit` _(entero, opcional)_ Número máximo de plantillas de memes a devolver. Por defecto es 20.

Esta herramienta recupera una lista de plantillas de memes populares que se pueden usar
para crear memes personalizados. Estas plantillas están ordenadas por popularidad
según cuántas veces han sido subtituladas.

Crear un meme personalizado utilizando una plantilla de Imgflip

Esta herramienta crea un meme personalizado añadiendo tu texto a una plantilla de meme existente. Puedes especificar texto en la parte superior e inferior, elegir fuentes y controlar el tamaño del texto.

**Parámetros**

- `template_id` _(string, requerido)_ El ID de la plantilla de meme a utilizar para la creación. Puedes obtener esto de get_popular_memes.
- `top_text` _(string, opcional)_ Texto para mostrar en la parte superior del meme. Déjalo vacío si no es necesario.
- `bottom_text` _(string, opcional)_ Texto para mostrar en la parte inferior del meme. Déjalo vacío si no es necesario.
- `font` _(Fuente, opcional)_ Familia de fuentes a utilizar para el texto. Por defecto es IMPACT.
- `max_font_size` _(entero, opcional)_ Tamaño máximo de fuente para el texto. Por defecto es 50.
- `no_watermark` _(booleano, opcional)_ Eliminar la marca de agua de Imgflip (función Premium). Por defecto es False.
