El kit de herramientas de Linear ofrece un conjunto simplificado de herramientas para interactuar con las funciones de seguimiento de problemas y gestión de equipos de Linear. Con este kit de herramientas, los usuarios pueden:

- Recuperar información detallada sobre un problema específico, incluyendo su estado, comentarios y dependencias relacionadas.
- Acceder a información completa del equipo, incluyendo miembros del equipo, roles y configuraciones.

Este kit de herramientas es ideal para usuarios que buscan leer y analizar datos de problemas y equipos dentro de Linear sin realizar modificaciones.

## Linear.GetIssue

<br />

          "/examples/integrations/toolkits/linear/get_issue_example_call_tool.js",
        ],
      },
    },
  ]}
/>

Obtener información detallada sobre un problema específico de Linear

**Parámetros**

- **issue_id** (`string`, requerido) El ID o identificador del problema de Linear (por ejemplo, 'FE-123', 'API-456') a recuperar.
- **include_comments** (`boolean`, opcional) Si se deben incluir comentarios en la respuesta. Por defecto es True.
- **include_attachments** (`boolean`, opcional) Si se deben incluir archivos adjuntos en la respuesta. Por defecto es True.
- **include_relations** (`boolean`, opcional) Si se deben incluir relaciones de problemas (bloqueos, dependencias) en la respuesta. Por defecto es True.
- **include_children** (`boolean`, opcional) Si se deben incluir sub-problemas en la respuesta. Por defecto es True.

## Linear.GetTeams

<br />
<TabbedCodeBlock
  tabs={[
    {
      label: "Llamar a la Herramienta Directamente",
      content: {
        Python: [
          "/examples/integrations/toolkits/linear/get_teams_example_call_tool.py",
        ],
        JavaScript: [
          "/examples/integrations/toolkits/linear/get_teams_example_call_tool.js",
        ],
      },
    },
  ]}
/>

Obtener equipos de Linear e información del equipo, incluyendo miembros del equipo

**Parámetros**

- **team_name** (`string`, opcional) Filtrar por nombre de equipo. Proporcionar un nombre de equipo específico (por ejemplo, 'Frontend', 'Product Web') o un nombre parcial. Utiliza esto para encontrar equipos específicos o verificar la membresía del equipo. Por defecto es None (todos los equipos).
- **include_archived** (`boolean`, opcional) Si se deben incluir equipos archivados en los resultados. Por defecto es False.
- **created_after** (`string`, opcional) Filtrar equipos creados después de esta fecha. Puede ser:
- Cadena de fecha relativa (por ejemplo, 'el mes pasado', 'esta semana', 'ayer')
- Cadena de fecha ISO (por ejemplo, 'YYYY-MM-DD')
  Por defecto es None (todo el tiempo).
- **limit** (`integer`, opcional) Número máximo de equipos a devolver. Mínimo 1, máximo 100. Por defecto es 50.
- **end_cursor** (`string`, opcional) Cursor para paginación - obtener equipos después de este cursor. Utiliza el 'end_cursor' de la respuesta anterior. Por defecto es None (comenzar desde el principio).

Por favor, consulta la documentación del [proveedor de autenticación de Linear](/home/auth-providers/linear) para aprender cómo configurar la autenticación.
