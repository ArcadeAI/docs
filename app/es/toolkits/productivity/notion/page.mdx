El kit de herramientas de Arcade Notion proporciona un conjunto preconstruido de herramientas para interactuar con Notion. Estas herramientas facilitan la creación de agentes y aplicaciones de IA que pueden:

- Obtener el contenido de una página
- Crear una nueva página
- Buscar páginas o bases de datos por título
- Obtener los metadatos de un objeto de Notion (página o base de datos)
- Obtener la estructura de carpetas de un espacio de trabajo

## NotionToolkit.SearchByTitle

<br />
<TabbedCodeBlock
  tabs={[
    {
      label: "Llamar a la herramienta con autorización del usuario",
      content: {
        Python: [
          "/examples/integrations/toolkits/notion/search_by_title_example_call_tool.py",
        ],
        JavaScript: [
          "/examples/integrations/toolkits/notion/search_by_title_example_call_tool.js",
        ],
      },
    },
  ]}
/>

Buscar títulos similares de páginas, bases de datos o ambos dentro del espacio de trabajo de Notion del usuario. Esta herramienta devuelve información mínima sobre los objetos coincidentes sin su contenido.

**Parámetros**

- **`query`** _(string, opcional)_
  Una subcadena para buscar dentro de los títulos de páginas y bases de datos. Si no se proporciona, se devuelven todos los elementos.
- **`select`** _(str, opcional)_
  Limitar la búsqueda solo a páginas o solo a bases de datos. Si se proporciona, debe ser uno de `page` o `database`. Por defecto, ambos.
- **`order_by`** _(str, opcional)_
  La dirección para ordenar los resultados de búsqueda por tiempo de última edición. Debe ser `ascending` o `descending`. Por defecto, `descending`.
- **`limit`** _(int, opcional)_
  El número máximo de resultados a devolver. Por defecto, 100. Usa -1 para sin límite.

## NotionToolkit.GetObjectMetadata

<br />
<TabbedCodeBlock
  tabs={[
    {
      label: "Llamar a la herramienta con autorización del usuario",
      content: {
        Python: [
          "/examples/integrations/toolkits/notion/get_object_metadata_example_call_tool.py",
        ],
        JavaScript: [
          "/examples/integrations/toolkits/notion/get_object_metadata_example_call_tool.js",
        ],
      },
    },
  ]}
/>

Obtener los metadatos de un objeto de Notion (página o base de datos) usando su título o ID. Debe proporcionarse uno de `object_title` o `object_id` (pero no ambos). Los metadatos devueltos incluyen el ID del objeto, marcas de tiempo, propiedades, URL y más.

**Parámetros**

- **`object_title`** _(string, opcional)_
  El título de la página o base de datos cuyos metadatos se deben recuperar.
- **`object_id`** _(string, opcional)_
  El ID de la página o base de datos cuyos metadatos se deben recuperar.
- **`object_type`** _(str, opcional)_
  El tipo de objeto para hacer coincidir el título. Si se proporciona, debe ser uno de `page` o `database`. Por defecto, ambos si no se proporciona.

## Autenticación

El kit de herramientas de Arcade Notion utiliza el [proveedor de autenticación de Notion](/home/auth-providers/notion) para conectarse a las cuentas de Notion de los usuarios.

Con la Plataforma en la Nube de Arcade, no hay nada que configurar. Tus usuarios verán `Arcade` como el nombre de la aplicación que solicita permiso.

Con una instalación autohospedada de Arcade, necesitas [configurar el proveedor de autenticación de Notion](/home/auth-providers/notion#configuring-notion-auth) con tus propias credenciales de aplicación de Notion.
