Buscar noticias con Google News.

**Parámetros**

- **`query`** _(string, requerido)_ Palabras clave para buscar artículos de noticias. Ej. 'Apple lanza un nuevo iPhone'.
- **`country_code`** _(string, opcional, por defecto `None`)_ Código de país de 2 caracteres para buscar artículos de noticias. Ej. 'us' (Estados Unidos). Por defecto `None` (buscar noticias globalmente).
- **`language`** _(string, opcional, por defecto 'en' Inglés)_ Código de idioma de 2 caracteres para buscar artículos de noticias. Ej. 'en' (Inglés). Por defecto 'en' (Inglés).
- **`limit`** _(int, opcional, por defecto `None`)_ Número máximo de artículos de noticias a devolver. Por defecto None (devuelve todos los resultados encontrados por la API).

## Autenticación

El kit de herramientas de Google News de Arcade utiliza [SerpAPI](https://serpapi.com/) para obtener datos de noticias de Google News.

- **Secreto:**
  - `SERP_API_KEY`: Tu clave API de SerpAPI.
    <Callout type="info">
      Configurar el secreto `SERP_API_KEY` solo es necesario si estás
      [autoalojando](/home/local-deployment/install/overview) Arcade. Si estás
      utilizando Arcade Cloud, el secreto ya está configurado para ti. Para gestionar tus
      secretos, ve a la [página de Secretos](https://api.arcade.dev/dashboard/auth/secrets) en el
      Dashboard de Arcade.
    </Callout>

## Parámetros por defecto

El idioma y el país son configurables a través de variables de entorno. Cuando se establecen, se utilizarán como predeterminados para las herramientas de Google News.

Proporcionar un valor diferente como argumento `language_code` o `country_code` en la llamada a la herramienta sobrescribirá el valor predeterminado.

**Idioma**

El código de idioma es un código de 2 caracteres que determina el idioma en el que la API buscará y devolverá artículos de noticias. Hay dos variables de entorno:

- `ARCADE_GOOGLE_LANGUAGE`: un valor predeterminado para todas las herramientas de búsqueda de Google. Si no se establece, por defecto es 'en' (Inglés).
- `ARCADE_GOOGLE_NEWS_LANGUAGE`: un valor predeterminado para las herramientas de búsqueda de noticias. Si no se establece, por defecto es `ARCADE_GOOGLE_LANGUAGE`.

Una lista de códigos de idioma soportados se puede encontrar [aquí](#languagecodes).

**País**

El código de país es un código de 2 caracteres que determina el país en el que la API buscará artículos de noticias. Hay dos variables de entorno:

- `ARCADE_GOOGLE_NEWS_COUNTRY`: un valor predeterminado para la herramienta `SearchNews`. Si no se establece, por defecto es `None` (buscar noticias globalmente).

Una lista de códigos de país soportados se puede encontrar [aquí](#countrycodes).
