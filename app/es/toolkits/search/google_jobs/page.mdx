Buscar ofertas de trabajo con Google Jobs.

**Parámetros**

- **`query`** _(string, requerido)_ Consulta de búsqueda. Proporciona un título de trabajo, nombre de la empresa y/o cualquier palabra clave en general que represente qué tipo de trabajos está buscando el usuario.
- **`location`** _(string, opcional, Predeterminado a `None`)_ Ubicación para buscar trabajos. Ej. 'Estados Unidos' o 'Nueva York, NY'. Predeterminado a None.
- **`language`** _(string, opcional, Predeterminado a 'en' Inglés)_ Código de idioma de 2 caracteres para usar en la búsqueda de Google Jobs.
- **`limit`** _(int, opcional, Predeterminado a 10)_ Número máximo de resultados a recuperar. Predeterminado a 10 (máx. soportado por la API).
- **`next_page_token`** _(string, opcional, Predeterminado a `None`)_ Token de la siguiente página para paginar resultados. Predeterminado a None (comenzar desde la primera página).

## Autenticación

El kit de herramientas de Arcade Google Jobs utiliza [SerpAPI](https://serpapi.com/) para obtener datos de trabajo de Google Jobs.

- **Secreto:**
  - `SERP_API_KEY`: Tu clave API de SerpAPI.
    <Callout type="info">
      Configurar el secreto `SERP_API_KEY` solo es necesario si estás
      [autoalojando](/home/local-deployment/install/overview) Arcade. Si estás
      usando Arcade Cloud, el secreto ya está configurado para ti. Para gestionar tus
      secretos, ve a la [página de Secretos
      ](https://api.arcade.dev/dashboard/auth/secrets) en el Dashboard de Arcade.
    </Callout>

## Parámetros predeterminados

El idioma es configurable a través de variables de entorno. Cuando se establecen, se utilizarán como predeterminados para las herramientas de Google Jobs.

Proporcionar un valor diferente como argumento `language` en una llamada a la herramienta sobrescribirá el valor predeterminado.

**Idioma**

El código de idioma es un código de 2 caracteres que determina el idioma en el que la API buscará y devolverá artículos de noticias. Hay dos variables de entorno:

- `ARCADE_GOOGLE_LANGUAGE`: un valor predeterminado para todas las herramientas de búsqueda de Google. Si no se establece, predetermina a 'en' (inglés).
- `ARCADE_GOOGLE_JOBS_LANGUAGE`: un valor predeterminado para las herramientas de búsqueda de trabajos. Si no se establece, predetermina a `ARCADE_GOOGLE_LANGUAGE`.

Una lista de códigos de idioma soportados se puede encontrar [aquí](#languagecodes).
