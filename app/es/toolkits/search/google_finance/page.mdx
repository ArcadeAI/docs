## GoogleFinance.GetStockSummary

Recupera información de resumen para una acción dada utilizando la API de Google Finance a través de SerpAPI. Esta herramienta devuelve el precio actual y el cambio de precio del día de negociación más reciente.

**Parámetros**

- **`ticker_symbol`** _(string, requerido)_: El símbolo de la acción, por ejemplo, 'GOOG'.
- **`exchange_identifier`** _(string, requerido)_: El identificador del mercado, por ejemplo, 'NASDAQ'.

## GoogleFinance.GetStockHistoricalData

Obtén datos históricos para una acción dada durante un período de tiempo definido. Esta herramienta devuelve los datos de precio y volumen de la acción junto con eventos clave cuando están disponibles.

**Parámetros**

- **`ticker_symbol`** _(string, requerido)_: El símbolo de la acción, por ejemplo, 'GOOG'.
- **`exchange_identifier`** _(string, requerido)_: El identificador del mercado, por ejemplo, 'NASDAQ' o 'NYSE'.
- **`window`** _(enum ([GoogleFinanceWindow](#googlefinancewindow)), opcional, por defecto a ONE_MONTH)_: Período de tiempo para los datos del gráfico.

## Auth

El kit de herramientas de Google Finance de Arcade utiliza la [SerpAPI](https://serpapi.com/) para obtener datos de acciones de Google Finance.

- **Secreto:**
  - `SERP_API_KEY`: Tu clave API de SerpAPI.
    <Callout type="info">
      Configurar el secreto `SERP_API_KEY` solo es necesario si estás
      [autoalojando](/home/local-deployment/install/overview) Arcade. Si estás
      utilizando Arcade Cloud, el secreto ya está configurado para ti. Para gestionar tus
      secretos, ve a la [página de Secretos](https://api.arcade.dev/dashboard/auth/secrets) en el
      Dashboard de Arcade.
    </Callout>

## GoogleFinanceWindow

Define el período de tiempo para obtener datos de acciones de Google Finance.

- **`ONE_DAY`**: Representa un período de tiempo de 1 día.
- **`FIVE_DAYS`**: Representa un período de tiempo de 5 días.
- **`ONE_MONTH`**: Representa un período de tiempo de 1 mes.
- **`SIX_MONTHS`**: Representa un período de tiempo de 6 meses.
- **`YEAR_TO_DATE`**: Representa el tiempo desde el inicio del año hasta la fecha actual.
- **`ONE_YEAR`**: Representa un período de tiempo de 1 año.
- **`FIVE_YEARS`**: Representa un período de tiempo de 5 años.
- **`MAX`**: Representa el período de tiempo máximo disponible.
