### Prerequisites

1. Create an <SignupLink linkLocation="docs:call-tools-directly">Arcade account</SignupLink>
1. Get an [Arcade API key](/home/getting-started/api-keys) and take note, you'll need it in the next steps.

### Install the Arcade client

<Tabs items={["Python", "JavaScript"]} storageKey="preferredLanguage">
<Tabs.Tab>
```python
pip install arcadepy
```
</Tabs.Tab>
<Tabs.Tab>
```javascript
npm install @arcadeai/arcadejs
```
</Tabs.Tab>
</Tabs>

### Instantiate and use the client

<Tabs items={["Python", "JavaScript"]} storageKey="preferredLanguage">
<Tabs.Tab>
Create a new script called `example.py`:
```python file=<rootDir>/examples/code/home/core-concepts/use-tools/call-tools-directly/quickstart.py
```
</Tabs.Tab>
<Tabs.Tab>
Create a new script called `example.mjs`:
```javascript file=<rootDir>/examples/code/home/core-concepts/use-tools/call-tools-directly/quickstart.mjs
```
</Tabs.Tab>
</Tabs>

### Run the code

<Tabs items={["Python", "JavaScript"]} storageKey="preferredLanguage">
