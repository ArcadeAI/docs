# Quickstart
 (Talk to sam and nate about what a quickstart should look like)


## Requirements

- Python 3.10+

## Install Arcade AI

```bash
pip install arcade-ai[arithmetic]
```

## Export OPENAI_API_KEY
```bash
export OPENAI_API_KEY="sk-..."
```


## Run the Actor

```bash
arcade run -t arcade_math "What is 2 + 2?"
```
