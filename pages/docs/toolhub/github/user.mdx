# Github User Tools

**Description:** Tools for user in the Github toolkit

## SetStarred

Star or un-star a GitHub repository.

For example, to star microsoft/vscode, you would use:
```
set_starred(owner="microsoft", name="vscode", starred=True)
```

### **Parameters:**

**owner**&nbsp;&nbsp;&nbsp;<span className="tool-param-type">string</span>&nbsp;&nbsp;&nbsp;<span className="tool-param-required">Required</span>&nbsp;&nbsp;&nbsp;  
The owner of the repository

**name**&nbsp;&nbsp;&nbsp;<span className="tool-param-type">string</span>&nbsp;&nbsp;&nbsp;<span className="tool-param-required">Required</span>&nbsp;&nbsp;&nbsp;  
The name of the repository
<hr style={{borderTop: '0.5px solid #888', margin: '10px 0'}} />
**starred**&nbsp;&nbsp;&nbsp;<span className="tool-param-type">boolean</span>&nbsp;&nbsp;&nbsp;<span className="tool-param-required">Required</span>&nbsp;&nbsp;&nbsp;  
Whether to star the repository or not
