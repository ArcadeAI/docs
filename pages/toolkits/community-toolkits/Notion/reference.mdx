### CreateSubpage

Create a new page under a parent page using its ID.

#### Parameters

- `parent_id`_(string, required)_ ID of the parent page
- `title`_(string, required)_ Title for the new page
- `content`_(string, required)_ Content in markdown format.

---

### CreatePageByParentTitle

Creates a new page under an existing Notion page or database.

#### Parameters

- `parent_title`_(string, required)_ Title of an existing page/database where the new page will be created, no need to include the page ID
- `title`_(string, required)_ Title for your new page
- `content`_(string, required)_ Content in markdown format.

---

### GetPageId

Finds and returns the ID of a Notion page by searching for its title. Only use when page ID is explicitly required.

#### Parameters

- `title`_(string, required)_ Title of the page to find
