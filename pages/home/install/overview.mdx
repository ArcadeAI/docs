# Installation

Explore the different installation options for Arcade AI.

- [Local](/home/install/local)
- [Docker](/home/install/docker)
- Kubernetes (coming soon)

## Supported Platforms

<Warning>
  The `arcade-engine` binary and containers will not be published to these
  registries until the public Beta release.
</Warning>

The `arcade-engine` binary can be downloaded through system package managers like `brew` or `apt-get` or directly from the links below.
