1. Get an Arcade API key
2. `pip install arcade-ai`
3. Install the toolkits you want to have available to Claude
    - `pip install arcade-google`
4. Download [Claude Desktop](https://claude.ai/download)
5. Open Claude Desktop
6. Open up the Claude menu on your computer and select "Settings"
7. Click on "Developer"
8. Click "Edit Config". This will create a configuration file at:
    - On Mac: `~/Library/Application Support/Claude/claude_desktop_config.json`
    - On Windows: `%APPDATA%\Claude\claude_desktop_config.json`
9. Open up the configuration file and replace the file contents with this:
```json
{
  "mcpServers": {
    "arcade-stdio": {
      "command": "bash",
      "args": [
        "-c",
        "export ARCADE_API_KEY=YOUR_ARCADE_API_KEY_HERE && /path/to/python /path/to/arcade serve --mcp"
      ]
    }
  }
}
```
Don't forget to replace `YOUR_ARCADE_API_KEY_HERE` with your actual Arcade API key and `/path/to/python` with the path to your Python interpreter and `/path/to/arcade` with the path to the Arcade package.
10. Restart Claude Desktop. Upon restarting, you should have access to the Arcade tools you installed.
