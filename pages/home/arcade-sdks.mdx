# Arcade SDKs

Arcade provides SDKs for several languages.  These SDKs make it easy to use Arcade's tools within your own agents and applications.

## Python SDK

Install with: `pip install arcadepy`

Learn more about the [Python SDK](https://github.com/ArcadeAI/arcade-py).

## JavaScript / TypeScript SDK

Install with: `npm install @arcadeai/arcadejs`

Learn more about the [JavaScript / TypeScript SDK](https://github.com/ArcadeAI/arcade-js).

## Go SDK

Install with: `go get -u 'github.com/stainless-sdks/arcade-engine-go'`

Learn more about the [Go SDK](https://github.com/ArcadeAI/arcade-go?tab=readme-ov-file).
