# Get started with Arcade by calling tools directly

Direct tool calling is useful in situations where you already know what tool is needed or when there is no prompt for an LLM to interpret and decide on the tool calling. It also gives you full control over what is executed by the Arcade Engine and how.

<Steps>

### Prerequisites

1. Create an <SignupLink utmParams={{
            utm_source: "docs",
            utm_medium: "page",
            utm_campaign: "call-tools-directly",
          }} linkLocation="docs:call-tools-directly">Arcade account</SignupLink>
1. Get an [Arcade API key](/home/api-keys) and take note, you'll need it in the next steps.

### Install the Arcade client

<Tabs items={["Python", "JavaScript"]}>
<Tabs.Tab>
```python
pip install arcadepy
```
</Tabs.Tab>
<Tabs.Tab>
```javascript
npm install @arcadeai/arcadejs
```
</Tabs.Tab>
</Tabs>

### Instantiate and use the client

<Tabs items={["Python", "JavaScript"]}>
<Tabs.Tab>
Create a new script called `example.py`:
```python file=<rootDir>/examples/code/home/use-tools/call-tools-directly/math_sqrt_directly.py
```
</Tabs.Tab>
<Tabs.Tab>
Create a new script called `example.js`:
```javascript file=<rootDir>/examples/code/home/use-tools/call-tools-directly/math_sqrt_directly.js
```
</Tabs.Tab>
</Tabs>

### Run the code

<Tabs items={["Python", "JavaScript"]}>
<Tabs.Tab>
```bash
python3 example.py
> The square root of 625 is 25
> Successfully starred the repository ArcadeAI/arcade-ai
```
</Tabs.Tab>
<Tabs.Tab>
```bash
node example.js
> The square root of 625 is 25
> Successfully starred the repository ArcadeAI/arcade-ai
```
</Tabs.Tab>
</Tabs>

Note that in this simple example, we exit the program while we wait for the user to authorize the tool.  In your real applications, you'll be doing something far more interesting - lean more about using Arcade with Frameworks in the [Frameworks](home/langchain/use-arcade-tools) section, or [how to build your own tools](/home/build-tools/create-a-toolkit).

</Steps>
