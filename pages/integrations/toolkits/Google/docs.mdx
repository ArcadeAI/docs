## Auth

**Toolkit Scopes:**


- `https://www.googleapis.com/auth/documents.readonly`
- `https://www.googleapis.com/auth/documents`






## Available Tools

| Tool Name                          | Description                                                    |
|------------------------------------|----------------------------------------------------------------|
| [GetDocumentById](#getdocumentbyid)                | Retrieve a Google Docs document by ID.                          |
| [InsertTextAtEndOfDocument](#inserttextatendofdocument)    | Insert text at the end of a Google Docs document.               |
| [CreateBlankDocument](#createblankdocument)                  | Create a new blank Google Docs document with a title.            |
| [CreateDocumentFromText](#createdocumentfromtext)            | Create a new Google Docs document with specified text content.   |

---

## GetDocumentById

Get the latest version of the specified Google Docs document.

**Auth:**

- **Scopes Required:**
  - `https://www.googleapis.com/auth/documents.readonly`

**Environment Variables:**

- None

**Parameters**

- **`document_id`** _(string, Required)_ The ID of the document to retrieve.


**Returns**

- _(dict)_ The document contents as a dictionary

---

## InsertTextAtEndOfDocument

Insert text at the end of an existing Google Docs document.

**Auth:**

- **Scopes Required:**
  - `https://www.googleapis.com/auth/documents`

**Environment Variables:**

- None

**Parameters**

- **`document_id`** _(string, Required)_ The ID of the document to update.
- **`text_content`** _(string, Required)_ The text content to insert into the document.

**Returns**

- _(dict)_ The response from the batchUpdate API as a dict

---

## CreateBlankDocument

Create a blank Google Docs document with the specified title.

**Auth:**

- **Scopes Required:**
  - `https://www.googleapis.com/auth/documents`

**Environment Variables:**

- None

**Parameters**

- **`title`** _(string, Required)_ The title of the blank document to create.

**Returns**

- _(dict)_ The created document's title, documentId, and documentUrl in a dictionary

---

## CreateDocumentFromText

Create a Google Docs document with the specified title and text content.

**Auth:**

- **Scopes Required:**
  - `https://www.googleapis.com/auth/documents`

**Environment Variables:**

- None

**Parameters**

- **`title`** _(string, Required)_ The title of the document to create.
- **`text_content`** _(string, Required)_ The text content to insert into the document.

**Returns**

- _(dict)_ The created document's title, documentId, and documentUrl in a dictionary
