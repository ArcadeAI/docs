## Available Tools

| Tool Name      | Description                                   |
|----------------|-----------------------------------------------|
| [CreateEvent](#createevent)   | Create a new event in Google Calendar.      |

---

## CreateEvent

Create a new event in the specified calendar.

**Auth:**

- **Scopes Required:**
  - `https://www.googleapis.com/auth/calendar.events`
  - `https://www.googleapis.com/auth/calendar`

**Environment Variables:**

- None

**Parameters**

- **`summary`** _(string, Required)_ The title of the event.
- **`start_date`** _(string, Required)_ The day the event starts.
- **`start_time`** _(string, Required)_ The time the event starts.
- **`end_date`** _(string, Required)_ The day the event ends.
- **`end_time`** _(string, Required)_ The time the event ends.
- **`calendar_id`** _(string, Optional, Defaults to `'primary'`)_ The ID of the calendar.
- **`description`** _(string, Optional)_ The description of the event.
- **`location`** _(string, Optional)_ The location of the event.
- **`visibility`** _(string, Optional)_ The visibility of the event (`'default'`, `'public'`, `'private'`, `'confidential'`).
- **`attendee_emails`** _(array of strings, Optional)_ Attendee emails.

---

## ListEvents

List events from the specified calendar within the given date range.

**Auth:**

- **Scopes Required:**
  - `https://www.googleapis.com/auth/calendar.readonly`

**Environment Variables:**

- None

**Parameters**

- **`min_day`** _(string, Required)_ Events ending on or after this day.
- **`min_time_slot`** _(string, Required)_ Events ending after this time.
- **`max_day`** _(string, Required)_ Events starting on or before this day.
- **`max_time_slot`** _(string, Required)_ Events starting before this time.
- **`calendar_id`** _(string, Optional, Defaults to `'primary'`)_ The ID of the calendar.
- **`max_results`** _(integer, Optional, Defaults to `10`)_ Maximum number of events to return.

---

## UpdateEvent

Update an existing event with provided details.

**Auth:**

- **Scopes Required:**
  - `https://www.googleapis.com/auth/calendar.events`
  - `https://www.googleapis.com/auth/calendar`

**Environment Variables:**

- None

**Parameters**

- **`event_id`** _(string, Required)_ The ID of the event to update.
- **`updated_start_day`** _(string, Optional)_ Updated start day.
- **`updated_start_time`** _(string, Optional)_ Updated start time.
- **`updated_end_day`** _(string, Optional)_ Updated end day.
- **`updated_end_time`** _(string, Optional)_ Updated end time.
- **`updated_calendar_id`** _(string, Optional)_ Updated calendar ID.
- **`updated_summary`** _(string, Optional)_ Updated event title.
- **`updated_description`** _(string, Optional)_ Updated description.
- **`updated_location`** _(string, Optional)_ Updated location.
- **`updated_visibility`** _(string, Optional)_ Updated visibility.
- **`attendee_emails_to_add`** _(array of strings, Optional)_ Attendee emails to add.
- **`attendee_emails_to_remove`** _(array of strings, Optional)_ Attendee emails to remove.
- **`send_updates`** _(string, Optional, Defaults to `'all'`)_ Who to notify about the update (`'all'`, `'externalOnly'`, `'none'`).

---

## DeleteEvent

Delete an event from Google Calendar.

**Auth:**

- **Scopes Required:**
  - `https://www.googleapis.com/auth/calendar.events`

**Environment Variables:**

- None

**Parameters**

- **`event_id`** _(string, Required)_ The ID of the event to delete.
- **`calendar_id`** _(string, Optional, Defaults to `'primary'`)_ The ID of the calendar.
- **`send_updates`** _(string, Optional, Defaults to `'all'`)_ Who to notify about the deletion (`'all'`, `'externalOnly'`, `'none'`).
